# 惣菜本部 業務システム

## ファイル構成
```
soozai-system-v2/
├── app.py        ← APIサーバー（Python Flask）
├── index.html    ← フロントエンド（ブラウザで開く）
├── soozai.db     ← データベース（SQLite・自動生成済み）
└── README.md
```

## 起動方法

### 1. APIサーバーを起動（毎回必要）
```bash
cd soozai-system-v2
python app.py
```
→ `http://localhost:5050` で起動します

### 2. ブラウザで index.html を開く
- `index.html` をダブルクリック、またはブラウザにドラッグ＆ドロップ
- Chrome / Edge 推奨

## 毎日の使い方（業務フロー）

### ① 出荷指示書
1. 左メニュー「出荷指示書」をクリック
2. 一週間分の数量を入力（先週からコピーも可能）
3. 「💾 保存」
4. 「🖨 ○/○を印刷」でその日分を印刷

### ② 出荷実績入力（メンバーが現場で入力）
1. 「✏️ ○/○の実績入力へ」ボタン、または左メニュー「出荷実績入力」
2. 計画数を参考に実績数を入力（差分が自動表示）
3. シフト欄にメンバー名と時間数を入力
4. 天気・別注文・メモを入力
5. 「💾 実績・シフトを保存」
6. 「📊 日報を生成する」→ 売上日報に自動移動

### ③ 売上日報の確認・修正
- 自動集計された売上・利益・人時売が表示
- 人件費を手修正して「更新・再計算」も可能
- 「🖨 印刷」でそのまま印刷

### ④ 週間献立（週1回）
- 左メニュー「週間献立」
- セルに直接入力して「保存」

### ⑤ 月次サマリ（月1回）
- 左メニュー「月次サマリ」
- 月を選択すると自動集計

## 売上計算ロジック
- 材料費 = 合計売上 × 50%
- 経費 = 固定 7,000円
- 人件費 = シフト入力後、手動で確認・修正
- 差引利益 = 売上 - 材料費 - 人件費 - 経費
- 人時売上 = 売上 ÷ 総人時

## 商品・出荷先の追加
- 「⚙️ 商品マスタ」から追加・編集
- 出荷先（西店/南店/市場/空港/医師会/NPO）は初期登録済み
